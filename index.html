Giới thiệu về css

Css là gì?
CSS là chữ viết tắt của Cascading Style Sheets, nó là một ngôn ngữ được sử dụng để tìm và định dạng lại các phần tử được tạo ra bởi các ngôn ngữ đánh dấu (HTML)

Tác dụng của Css
CSS sẽ tìm dựa vào các vùng chọn, vùng chọn có thể là tên một thẻ HTML, tên một ID, class hay nhiều kiểu khác. Sau đó là nó sẽ áp dụng các thuộc tính cần thay đổi lên vùng chọn đó.

Phân biệt inline Css, internal Css, external Css

inline css
Css được biết trong thẻ mở của các thẻ html

inline css
Css được biết trong thẻ mở của các thẻ html

internal css

external css
Css được viết trong file .css
